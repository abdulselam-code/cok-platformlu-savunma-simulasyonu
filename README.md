# Çok Platformlu Savunma Sistemleri 2B Yörünge Simülasyonu

Bu proje, Grafik Programlama dersi için hazırlanan eğitim amaçlı ve kurgusal bir 2B fizik ile grafik programlama simülasyonudur.

## Kullanılan Teknoloji

- Python 3.13
- PySide6

## Kurulum

```bash
python -m pip install -r requirements.txt
```

## Çalıştırma

```bash
python main.py
```

## Mevcut Durum

Şu an yalnızca başlangıç arayüzü tamamlanmıştır.

## Gelecek Modüller

Gelecek aşamalarda tank, uçak, gemi ve denizaltı simülasyonları eklenecektir.
